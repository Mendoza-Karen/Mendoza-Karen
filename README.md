# 🎆 Karen Mujica Mendoza

**`Analista de Dados Jr.`**

Oi, tudo bem? Meu nome de Karen. 👋  

Sou formada em Ciência da Computação e apaixonada por dados, tecnologia e resolução de problemas.

Desde o início da minha trajetória, sempre tive interesse por tecnologia e análise de informações. Ao longo da graduação, tive contato com programação, lógica e desenvolvimento de sistemas, o que despertou ainda mais meu interesse por trabalhar com dados e gerar insights a partir deles.  

Buscando me especializar nessa área, concluí uma formação em Data Analytics pela TripleTen, onde desenvolvi diversos projetos práticos com dados reais. Durante essa jornada, me aprofundei em todo o ciclo de análise de dados — desde a coleta e limpeza até a análise exploratória, modelagem e visualização.  

Ao longo dos meus projetos, trabalhei com:  
- Análise Exploratória de Dados (EDA)  
- Criação e monitoramento de KPIs  
- Testes A/B e análise estatística  
- Análise de desempenho operacional (como projetos com dados de call center)  
- Limpeza, transformação e estruturação de dados  
- Construção de dashboards interativos  

Tenho experiência com ferramentas e tecnologias como **Python (Pandas, NumPy, Matplotlib, Seaborn), SQL, Power BI e Tableau**, além de familiaridade com bancos de dados como MySQL, SQL Server e MongoDB.  
 

Sou uma pessoa organizada, proativa e com grande interesse em aprendizado contínuo. Gosto de transformar dados em informações claras e úteis para apoiar decisões e gerar valor para o negócio.  

📊 Atualmente, estou focada em evoluir como Analista de Dados e construir soluções que realmente façam diferença.
